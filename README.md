# Data Structures in C++

This repository contains implementations of fundamental **Data Structures** in C++ for learning and practice.

---

# 📌 Included Data Structures

## 1. Singly Linked List

A simple implementation of a **Singly Linked List** using OOP.

### Features:
- Insert at beginning
- Insert at end
- Insert at specific position
- Delete a node
- Search for an element
- Display the list
- Get size of the list

---

## 2. Queue (FIFO - First In First Out)

Implementation of **Queue** using two approaches:
- Dynamic Array
- Linked List

### Features:
- Enqueue (Insert element)
- Dequeue (Remove element)
- Peek (Front element)
- Get rear element
- Display elements
- Search element
- Count elements
- Clear queue

---

## 3. Stack (LIFO - Last In First Out)

Implementation of **Stack** using three approaches:
- Static Stack (fixed array)
- Dynamic Stack (resizable array)
- Linked Stack (linked list)

### Features:
- Push (Insert element)
- Pop (Remove element)
- Peek (Top element)
- Display stack
- Get size
- Copy stack (Linked Stack)

---

## 4. Binary Search Tree (BST)

A full implementation of **Binary Search Tree**.

### Features:
- Insert nodes
- Search for a value
- Delete nodes (leaf / one child / two children)
- Traversals:
  - Inorder
  - Preorder
  - Postorder
- Find minimum and maximum values
- Tree statistics:
  - Total nodes
  - Leaf nodes
  - Internal (parent) nodes
  - Height of tree / node
  - Depth of a node
